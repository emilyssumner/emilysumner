Patents#

This portfolio spans 28 patents (4 granted, more pending) across autonomous vehicle systems, human-machine interaction, driver state monitoring, shared control, and cognitive modeling. The work clusters around several themes: personalizing driver interfaces using inferred cognitive states, enabling accessible in-vehicle interactions for visually impaired riders, shared-autonomy and coaching frameworks for skill acquisition, and computational models of human preference and decision-making.
